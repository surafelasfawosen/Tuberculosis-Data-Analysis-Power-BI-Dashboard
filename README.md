## 📋 Project Overview

This project presents an **interactive Power BI dashboard** for analyzing global Tuberculosis (TB) data from **1990 to 2013**,Forecast future TB trends with update using 2024 data from Kaggle. 

The dashboard explores TB cases, deaths, incidence rates, and the burden of HIV-TB co-infection across regions and countries, while also providing future forecasts.

---

## Objectives

- Analyze historical trends of TB cases and deaths (1990–2013)
- Understand the impact of HIV on TB mortality
- Compare TB burden across different WHO regions and countries
- Forecast future TB trends using Power BI’s built-in forecasting tool
- Compare forecasted values with real 2024 data
- Deliver clear data storytelling with insights and recommendations

---

##  Data Preparation & Cleaning

- Loaded and cleaned TB dataset in Power Query Editor
- Renamed columns for better readability
- Converted data types (Integer & Decimal)
- Handled missing values using Mean and Mode imputation
- Retained key features: Total TB Cases, TB Deaths, Incidence per 100k, Mortality per 100k, HIV-TB Deaths, Case Detection Rate, etc.

##  Key Insights & Data Storytelling

### Past Trends (1990–2012)
Global TB deaths declined due to increased funding and better treatment programs, but HIV-associated TB remained a major challenge in high-burden countries.

### Present Status (2013)
HIV-related TB deaths were disproportionately high in Africa (especially South Africa), highlighting the severe burden of co-infection in regions with limited access to treatment.

### Future Outlook
The forecast predicts a **gradual decline** in TB cases and deaths if current efforts continue. However, without targeted interventions in high-burden regions like Sub-Saharan Africa, progress may stall.
